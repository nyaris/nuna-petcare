# 🐾 Nuna — A Gentle Pet Care Experience

Nuna is a modern, beautifully crafted pet care app designed to help you nurture your pets with simplicity, warmth, and intention.

Built with **SwiftUI** and guided by a scalable **Design System architecture**, Nuna focuses on creating a delightful and intuitive experience while showcasing best practices in declarative UI, state management, and reusable components.

---

## ✨ Features

* 🐶 **Pet Dashboard** — View all your pets in a clean, card-based interface
* 💜 **Pet Profiles** — Track mood, activities, and daily care
* 📝 **Activity Logging** — Register feeding, walks, and playtime
* ➕ **Add New Pets** — Simple and elegant onboarding flow
* 🎨 **Theming & Design System** — Consistent UI powered by reusable tokens and components

---

## 🎨 Design Philosophy

Nuna embraces a **soft, calming visual language** inspired by pastel tones and minimal interfaces.  
Every detail is crafted to feel light, friendly, and emotionally engaging — turning everyday pet care into a pleasant experience.

---

## 🧱 Architecture

* **SwiftUI-first approach**
* **MVVM pattern**
* Modular and scalable **Design System**
* Reusable components (Buttons, Cards, Inputs)
* Centralized design tokens (Colors, Typography, Spacing)

---

## 🚀 Purpose

This project was created as a hands-on exploration of:

* Transitioning from UIKit to SwiftUI
* Building scalable UI systems
* Designing production-ready components
